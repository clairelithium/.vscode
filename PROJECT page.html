PROJECT 

 
For this project, I used HTML, CSS, and Python to create the tic-tac-toe game and website portfolio. Of course, I enjoyed creating this project. Even though it is a simple project,
I do believe that it presents a decent amount of experience I have with coding, and shows the fun I had while organizing it. 

However, there is certainly some struggles I went through while making this project. It took me quite a long time to figure out how to attach pages to another page, and link them. 
Additionally, I encountered a huge conflict when bunch of my texts were overlapping each other, which I was able to fix with the code "margin-..." and "text-align".
Furthermore, when I created my website, it was clear that I needed to place images to make it seem more appealing. Although, when I placed in the photos, it took me a long 
time to figure out how to adjust the photos, align them to the texts, and make it look pleasant! 


Overall, I'm quite proud of this project because I believe I was able to initiate a wonderful output and enjoy the process. In spite of the fact that it took some time
(which I ended up having to extend the time finishing this project until the day right before my birthday, July 1st!), to organize the project to produce a desireable output, 
I am happy that I was able to finish this project and learn some steps in the time-being! Some of the skills and code that I learnt and used for these two projects will be helpful
in the future, and I definitely know that I will use it for other circumstances, rather than just a website portfolio or a simple game, but even more! As said before, I am
intrigued in computer science and I will likely continue to learn more about it with high fascination!
